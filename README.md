# CPU Scheduling Simulator

A Python-based simulator that demonstrates different CPU scheduling algorithms and visualizes their execution using Gantt charts.

## Algorithms Implemented

* First Come First Serve (FCFS)
* Shortest Job First (SJF)
* Shortest Remaining Time First (SRTF)
* Priority Scheduling
* Round Robin

## Features

* Visual Gantt chart generation
* Comparison of average waiting time
* Comparison of turnaround time
* Identifies best scheduling algorithm based on performance

## Tech Stack

* Python
* Matplotlib
* Streamlit (for web interface)

## Run Locally

Install dependencies:

pip install streamlit matplotlib

Run the application:

streamlit run os_app.py
